# O que é Python?

Python é uma linguagem de programação de **alto nível** (próxima da linguagem humana), *interpretada* (executada linha por linha), de propósito geral e multiparadigma. Criada por Guido van Rossum e lançada em 1991, é uma das linguagens mais populares do mundo.

Sua principal filosofia de design foca na **legibilidade do código**, usando uma sintaxe clara e simples que se assemelha à língua inglesa. Isso a torna uma excelente escolha para iniciantes.

## Principais Características

* **Sintaxe Simples e Legível:** Utiliza indentação (espaços em branco) para definir blocos de código, em vez de chaves ou palavras-chave, o que força um código visualmente limpo.
* **Interpretada:** O código não precisa ser compilado antes da execução, o que agiliza o desenvolvimento e a depuração.
* **Multiparadigma:** Suporta múltiplos estilos de programação, como Programação Orientada a Objetos (POO), Programação Funcional e Programação Procedural.
* **Tipagem Dinâmica:** Os tipos de variáveis são verificados em tempo de execução, não sendo necessário declará-los explicitamente.
* **Vasta Biblioteca Padrão:** Vem com uma grande coleção de módulos e funções prontos para uso, permitindo realizar uma variedade de tarefas (como manipulação de texto, conexões de rede, etc.) sem instalar pacotes extras.
* **Multiplataforma:** Um código Python escrito em um sistema operacional (como Windows) pode rodar em outros (como macOS ou Linux) sem modificações.
* **Versátil:** É amplamente utilizada em diversas áreas, incluindo:
    * Desenvolvimento Web (backend) com frameworks como Django e Flask.
    * Ciência de Dados, Análise de Dados e Machine Learning.
    * Automação de tarefas e scripts.
    * Desenvolvimento de software e prototipagem.
