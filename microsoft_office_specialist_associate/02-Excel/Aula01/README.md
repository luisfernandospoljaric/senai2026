## Operações

| Fórmula                     | Exemplos                                                                 |
|----------------------------|---------------------------------------------------------------------------|
| `=SOMA(A1:A10)`            | Ex: Soma os valores das células de A1 até A10.                                |
| `=MÉDIA(B2:B12)`           | Ex: Calcula a média dos valores entre B2 e B12.                              |
| `=MÍNIMO(E1:E50)`          | Ex: Retorna o menor valor do intervalo E1 a E50.                             |
