## O que é uma Linguagem de Programação?

Uma **linguagem de programação** é um meio formal utilizado para escrever instruções que o computador pode entender e executar.  
Ela funciona como uma ponte entre o raciocínio humano e a máquina.

Exemplos de linguagens:
- Python
- Java
- C
- C++
- JavaScript
- PHP

---

## Princípios das Linguagens de Programação

### Abstração
Permite ao programador resolver problemas sem se preocupar com os detalhes internos de funcionamento do sistema.

### Precisão
O computador executa exatamente o que foi escrito, sem interpretações subjetivas.

### Automação
Tudo que pode ser descrito em passos lógicos pode ser automatizado por meio de um programa.

### Eficiência
Relaciona-se ao uso otimizado de recursos como memória e tempo de processamento.

### Portabilidade
Capacidade de um programa ser executado em diferentes sistemas operacionais.

---

## Características das Linguagens de Programação

### Sintaxe
Define as regras de escrita da linguagem.

### Semântica
Define o significado das instruções escritas.

### Tipagem
Pode ser:
- **Estática**: tipo definido antes da execução (ex: C, Java)
- **Dinâmica**: tipo definido durante a execução (ex: Python, JavaScript)

### Legibilidade
Facilidade de leitura e compreensão do código.

### Confiabilidade
Capacidade da linguagem de evitar erros e lidar com falhas.

## Tipos de Linguagens de Programação

### Linguagens Compiladas

São linguagens cujo código-fonte é totalmente traduzido para código de máquina antes da execução.

**Fluxo de execução:**
Código-fonte → Compilador → Executável → Execução

**Características:**
- Alto desempenho
- Execução mais rápida
- Necessidade de recompilação após alterações

**Exemplos:**
- C
- C++
- Go
- Rust

---

### Linguagens Interpretadas

São linguagens executadas linha por linha por um interpretador, sem gerar um executável final.

**Fluxo de execução:**

Código-fonte → Interpretador → Execução


**Características:**
- Maior flexibilidade
- Facilidade de aprendizado
- Execução geralmente mais lenta

**Exemplos:**
- Python
- JavaScript
- PHP
- Ruby

---

## Comparação: Compilada vs Interpretada

| Característica        | Compilada           | Interpretada        |
|----------------------|--------------------|--------------------|
| Forma de execução    | Antes da execução  | Linha por linha    |
| Velocidade           | Alta               | Menor              |
| Flexibilidade        | Menor              | Maior              |
| Facilidade de uso    | Média              | Alta               |
| Exemplos             | C, C++             | Python, JS         |

---

## Observação Importante

Algumas linguagens são consideradas **híbridas**, pois utilizam tanto compilação quanto interpretação.

**Exemplos:**
- **Java**: compila para bytecode e executa na JVM
- **Python**: gera bytecode, mas é interpretado
